Python Data Types Tutorial

Updated: June 14, 2026 | Reading time: 8 minutes

Introduction to Data Types

Data types define what kind of value a variable can hold. Python supports various built-in data types that help you work with different kinds of information.

Strings (str)

Strings are sequences of characters enclosed in quotes:

name = "John"          # double quotes
city = 'New York'      # single quotes
sentence = """Multi
line
string"""

String Methods:

  • .upper() - Convert to uppercase
  • .lower() - Convert to lowercase
  • .strip() - Remove whitespace
  • .split() - Split into list
  • .replace() - Replace characters

Numbers - Integers (int)

Integers are whole numbers without decimals:

age = 25
count = -10
zero = 0
large_number = 1_000_000  # underscores for readability

Numbers - Floats (float)

Floats are numbers with decimal points:

price = 19.99
pi = 3.14159
scientific = 1.5e-10
negative = -3.14

Booleans (bool)

Booleans represent True or False values:

is_student = True
is_graduated = False
is_active = 1 > 0  # Result: True
print(type(True))  #

Lists (list)

Lists store multiple items in a single variable, ordered and changeable:

fruits = ["apple", "banana", "cherry"]
numbers = [1, 2, 3, 4, 5]
mixed = [1, "hello", 3.14, True]

# Access items
first = fruits[0]  # "apple"
last = fruits[-1]  # "cherry"

# Modify items
fruits[0] = "orange"

# Add items
fruits.append("grape")
fruits.extend(["kiwi", "mango"])

Tuples (tuple)

Tuples are like lists but immutable (cannot be changed):

colors = ("red", "green", "blue")
coordinates = (10, 20)

# Access items
first_color = colors[0]  # "red"

# Tuples cannot be modified
# colors[0] = "yellow"  # This will cause an error

Dictionaries (dict)

Dictionaries store data as key-value pairs:

person = {
    "name": "Alice",
    "age": 30,
    "city": "New York",
    "is_student": False
}

# Access values
name = person["name"]     # "Alice"
age = person.get("age")   # 30

# Add/modify items
person["email"] = "alice@example.com"
person["age"] = 31

# Remove items
del person["city"]
person.pop("is_student")

Sets (set)

Sets are unordered collections of unique items:

colors = {"red", "green", "blue"}
numbers = {1, 2, 3, 3, 4}  # Duplicates removed

# Check membership
if "red" in colors:
    print("red is in the set")

# Add items
colors.add("yellow")

# Remove items
colors.remove("green")

Type Checking

Use the type() function to check data types:

print(type("hello"))      # 
print(type(42))           # 
print(type(3.14))         # 
print(type(True))         # 
print(type([1, 2, 3]))    # 
print(type({1, 2, 3}))    # 
print(type({"a": 1}))     #

Key Takeaways

  • Strings store text data
  • Numbers can be integers or floats
  • Booleans are True/False values
  • Lists are mutable ordered collections
  • Tuples are immutable ordered collections
  • Dictionaries store key-value pairs
  • Sets store unique unordered items
  • Use type() to check data types
